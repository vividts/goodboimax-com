# GoodBoiMax.com 🐾

> Premium Parts for Premium Bois™

A whimsical mock e-commerce site where you assemble a Good Boi from scratch.
No real purchases. All fun.

## Running locally

Just open `index.html` in a browser. No build step required.

Or serve with any static file server:

```
npx serve .
python3 -m http.server 8080
```

## Structure

```
index.html       # SPA shell
css/style.css    # All styles
js/app.js        # State machine + rendering
js/data.js       # Product catalog
assets/dog.svg   # Layered cartoon dog
```
